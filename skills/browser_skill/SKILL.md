# Skill `browser_skill`

## Khi nào dùng

Dùng skill này khi cần:

- mở trang web
- tìm kiếm thông tin trên internet
- truy cập ERP/CRM/web nội bộ
- click, nhập liệu, lấy dữ liệu từ giao diện web

## Cách gọi

Chỉ dùng `exec` để chạy:

```bash
python D:/CodeAiTanPhat/TPE_OpenClaw/skills/browser_skill/browser_skill.py "{\"task\":\"Tìm giá cầu nâng 2 trụ trên web và tóm tắt giúp tôi\"}"
```
